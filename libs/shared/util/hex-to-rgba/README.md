## Description

Angular pipe to convert a hex color to rgba.

## Usage

```html
<div [style.backgroundColor]="hexColor | hexToRgba: 0.5"></div>
```

## Running unit tests

Run `nx test hex-to-rgba` to execute the unit tests.

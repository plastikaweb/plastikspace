## Description

This library contains shared **types, interfaces, and entities** used across the authentication system.

## Contents

- **Types and Interfaces**: Common TypeScript types and interfaces for authentication data structures.
- **Models**: Entity definitions for user authentication states and related data.

## Running unit tests

Run `nx test shared-auth-entities` to execute the unit tests.
